## 2026-04-02 [CTO] → index.html 全体レビュー

### 1. 筆記体フォント未適用

- **指摘**: CCO方針「装飾は筆記体（英字）」が未対応。セクションラベル（APPROACH, Course等）がゴシック体のまま
- **理由**: Instagramのトーンとの一貫性（ゴールド装飾ライン + 筆記体）
- **対応**: Google Fonts に Dancing Script を追加。`.label` クラスに `font-family: 'Dancing Script', cursive` を適用。font-size も 10px → 16px に調整してスクリプト体が映えるサイズに
- **ステータス**: 反映済み

### 2. 「赤もちらっと」が未反映

- **指摘**: CCOデザイン方針「薄ピンクベース、赤もちらっと」の「赤」要素がページ内に存在しない
- **理由**: Instagramでもマゼンタピンクが強調テキストに使われている。ブランドカラーの一部
- **対応**: `#c94060`（落ち着いた赤）を3箇所に適用。(1) ヒーローh1の「可愛く」(em要素)、(2) statementの「綺麗になって、」、(3) instructorの「あなたは、まだまだ可愛くなれる」
- **ステータス**: 反映済み

### 3. Unsplash外部画像のハードコード

- **指摘**: 共感セクション（empathy）に Unsplash の外部画像URLが5枚ハードコードされていた
- **理由**: 写真素材はCEOが後日配置する方針。外部URLに依存すると、(a) ライセンス問題、(b) 読み込み速度劣化、(c) URL変更時の表示崩れリスクがある
- **対応**: 全5枚をプレースホルダー（`<div class="card-img placeholder-img">`）に置換。CSS `.placeholder-img` を追加
- **ステータス**: 反映済み

### 4. ゴールドアクセントの強化

- **指摘**: `.thin-line` のゴールド `#d4a97a` が小さく目立たない
- **理由**: Instagramではゴールドの枠線・装飾が特徴的なアクセント
- **対応**: 幅を 30px → 40px に拡大。`linear-gradient` でゴールドのグラデーションを適用
- **ステータス**: 反映済み

### 5. ボタンの角丸統一

- **指摘**: 全ボタン（hero-cta, course-cta, cta .btn, line-float, badge）が `border-radius: 2px` でほぼ角ばっていた
- **理由**: 「癒されて可愛くなれる」イメージにはもう少し柔らかさが欲しい。ただし丸すぎると幼くなるため控えめに
- **対応**: 全ボタンを `border-radius: 4px` に統一
- **ステータス**: 反映済み

### 6. スムーススクロール未設定

- **指摘**: ナビのアンカーリンクがジャンプ移動（カクッと飛ぶ）
- **理由**: UXとして滑らかなスクロールの方が「癒し」のトーンに合う
- **対応**: `html { scroll-behavior: smooth; }` を追加
- **ステータス**: 反映済み

### 7. ナビに「お客様の声」リンク追加

- **指摘**: Voicesセクション（`#voice`）へのナビリンクが存在しなかった
- **理由**: 社会的証明は体験レッスン申込の意思決定に直結する。ナビから1クリックでアクセスできるべき
- **対応**: ナビに `<li><a href="#voice">お客様の声</a></li>` を追加
- **ステータス**: 反映済み

## 2026-04-02 [CCO] → index.html デザイン最終チェック

### 8. CTAセクションの筆記体ラベル視認性

- **指摘**: "Trial Lesson" ラベルが `rgba(255,255,255,0.5)` で背景 #c4788a に対して薄すぎる
- **理由**: 50代〜60代ターゲットの視認性を考慮。装飾ラベルであっても読めないのは不適切
- **対応**: opacity を 0.5 → 0.7 に変更
- **ステータス**: 反映済み

### 9. 共感セクションのラベルがDancing Script（英字フォント）で日本語表示

- **指摘**: `.label` クラスに日本語テキスト「こんなお悩みありませんか」が入っていた。Dancing Script は英字フォントなのでフォールバックされ、他セクション（Approach, Course等）との統一感が崩れる
- **理由**: 全セクションの `.label` は筆記体英字で統一すべき。日本語の問いかけはh2見出しに含める
- **対応**: ラベルを "Worries" に変更。「こんなお悩みありませんか。」はh2の冒頭に移動
- **ステータス**: 反映済み

### 10. 赤文字（#c94060）のインラインスタイルをクラス化

- **指摘**: statementセクションとinstructorセクションの赤文字が `style="color:#c94060"` のインラインスタイルだった
- **理由**: 色の一元管理。赤の色味を変更する場合に全箇所を探す必要がなくなる
- **対応**: `.accent-red` クラスを追加し、インラインスタイルを置換
- **ステータス**: 反映済み

### CCO最終判定

**OK（デザインチェック合格）** -- 以下の条件付き:
- 写真素材が入った段階で、色味とのバランスを再確認する可能性あり
- CTA先のLINE URL差し替えは技術対応のみ（デザイン影響なし）
